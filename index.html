imgshare



Imgshare is a small OSX gui app created with Python using the imgur API. When a desktop screenshot is taken, it automatically uploads the image to your imgur account and provides a link by either sending it to the clipboard, or storing it for later use.



Find it on Github

EPA Search



EPA search is another Python program which used OracleSQL and R, the statistical programming language. Primarily a practice in database creation and use, it allowed for the construction of simple SQL queries in order to visualize the companies with the most EPA violations



Find it on Github
